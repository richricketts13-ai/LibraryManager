
📖 Overview

This is a simple Java console application that simulates a basic library system.
It allows users to manage books and members, and demonstrates object-oriented programming concepts.

⸻

🧱 Features
	•	Add and manage books
	•	Add and manage library members
	•	Basic library operations (check out / return books)
	•	Uses multiple classes working together
	•	Demonstrates OOP concepts like:
	•	Classes and objects
	•	Constructors
	•	Encapsulation

⸻

🗂 Project Structure
	•	Book.java – Represents a book in the library
	•	Member.java – Represents a library member
	•	Library.java – Handles the main logic and storage
	•	LibraryApp.java – Entry point of the program

⸻

▶️ How to Run
	1.	Open the project in your Java environment
	2.	Compile the files:

  javac *.java


3.	Run the program:

 java LibraryApp
